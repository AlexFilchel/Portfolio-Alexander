# Portfolio Alexander

Portfolio frontend en React + TailwindCSS con experiencia visual inspirada en Windows 11.

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

## Qué incluye

- Escritorio estilo Windows 11 con wallpaper, glassmorphism y taskbar centrada.
- Ventanas reutilizables con abrir, cerrar, minimizar y maximizar/restaurar.
- Apps: Sobre mí, Descargar CV, Cómo funciona la página y Mis proyectos.
- Datos mock separados y arquitectura modular para escalar.
